# Connect 4 Game

This project implements the Connect 4 game using HTML, CSS, JavaScript (JQquery), and the Bootstrap framework.

## Description

Connect 4 is a classic board game where players take turns dropping colored discs into a grid. The objective is to connect four of one's own discs of the same color vertically, horizontally, or diagonally before your opponent does.

## Features

- **Interactive Gameplay**: Players can select the size of the game board and who starts the game.
- **Single Player Mode**: Option to play against the computer.
- **Responsive Design**: The game is designed to be responsive and playable on various screen sizes.
- **Win Detection**: Automatically detects when a player wins by connecting four discs in a row.
- **Reset Button**: Allows players to reset the game and start over.

## How to Play

1. Select the size of the game board by entering a number between 5 and 10.
2. Choose who starts the game: either the player or the computer.
3. Click on the yellow or red disc to choose your color.
4. Click on a column in the game board to drop your disc.
5. Try to connect four discs of your color vertically, horizontally, or diagonally before your opponent does.

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap
- JQquery
